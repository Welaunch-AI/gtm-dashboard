# WeLaunch Client Portal

Private online workspace for **WeLaunch** (marketing/agency business) to collaborate with its clients. A shared hub where the agency and each client meet online — replacing scattered emails, spreadsheets, and separate tools.

## Product vision

A branded client portal: one clean, premium place where a marketing agency delivers results, communicates, and collaborates with each client.

**V1 scope:** Agency mostly enters data by hand. Only AI voice-call data flows in automatically. Later versions will automate more integrations.

---

## Two sides

### Agency team (admins)

- See everything: every client, every project, every number.
- All client-facing features, plus internal-only areas:
  - Scratch space for drafting marketing content
  - Activity log (who did what)
  - Tools to add new clients
  - Ability to edit/update every client's data

### Clients

- See only their own data — results, tasks, content to approve, etc.
- Cannot see other clients.

---

## Client portal features

| Area | Purpose |
|------|---------|
| **Dashboard** | Marketing performance: leads, meetings booked, campaign results |
| **Tasks** | What the agency is working on; shared weekly/monthly goals |
| **Tickets** | Raise requests or report issues (help desk) |
| **Marketing calendar** | Review social posts / drafted content; approve |
| **Knowledge base** | Shared docs and saved logins (mini password vault) |
| **Voice agent calls** | Listen to and read transcripts of AI phone calls on their behalf |
| **Demo tracker** | Simple CRM of leads/companies they're chasing |
| **Channels** | Slack-style chat with the agency team |

---

## Agency-only (internal)

- Content drafting scratch space
- Activity log
- Client onboarding / management
- Full edit access to all client data
